Generate a simulated dataset with G classes, n rows, and p features.

Each class is drawn from a normal distribution with a shifted mean. The resulting data is sorted by class label.

Parameters
nNumber of rows (observations).
pNumber of feature columns.
GNumber of classes (must be > 1).
rngRandom number generator.
