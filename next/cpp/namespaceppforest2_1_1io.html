◆ read_csv()

stats::DataPacket ppforest2::io::read_csv ( const std::string & filename)

Read a CSV file into a DataPacket.

Assumes the last column is the response variable (class label as string) and all preceding columns are numeric features. String labels are mapped to contiguous integer codes starting at 0.

Parameters

Returns
A DataPacket containing the feature matrix and response vector.
