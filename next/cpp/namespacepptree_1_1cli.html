Parse vars from a string (CLI input).

Formats:

  • "1/3" → fraction, is_proportion = true, value = 0.333...
  • "0.5" → decimal, is_proportion = true, value = 0.5
  • "5" → count, is_proportion = false, value = 5

◆ print_variable_importance()

void pptree::cli::print_variable_importance ( pptree::io::Output & out,
const VariableImportance & vi,
int max_rows = 20 )

Print a ranked variable importance table to stdout.

Columns: rank, variable name, sigma (scale), VI2 (projections), VI3 (weighted), VI1 (permuted). Rows are sorted by VI2 descending. The VI1 column is omitted when its vector is empty. At most max_rows rows are printed.

Parameters
outOutput context.
viVariable importance results.
max_rowsMaximum number of rows to print (0 = all).

For each scenario, spawns pptree evaluate as a child process with the appropriate flags, reads its JSON output, and collects results.
