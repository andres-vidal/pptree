Detailed Description

JSON serialization and deserialization for pptree models.

Uses nlohmann/json. Provides to_json() for serializing trees, forests, confusion matrices, and variable importance to JSON, and *_from_json() for deserializing them back. Also provides ostream operators for convenient text output.
