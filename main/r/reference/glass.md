# Glass Dataset

Contains measurements 214 observations of 6 types of glass; defined in
terms of their oxide content.

- Type:

  has 6 types of glasses

- X1:

  refractive index

- X2:

  Sodium (unit measurement: weight percent in corresponding oxide).

- X3:

  Magnesium

- X4:

  Aluminum

- X5:

  Silicon

- X6:

  Potassium

- X7:

  Calcium

- X8:

  Barium

- X9:

  Iron

## Usage

``` r
data(glass)
```

## Format

A data frame with 214 rows and 10 variables
