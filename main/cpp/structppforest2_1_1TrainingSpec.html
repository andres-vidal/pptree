Detailed Description

Training configuration for projection pursuit trees and forests.

Composes seven strategies (projection pursuit, variable selection, split cutpoint, stop rule, binarization, partition, leaf) together with forest-level parameters (size, seed, threads, max retries).

TrainingSpec is a concrete class — new strategies are plugged in via the builder, not by subclassing:

// Single tree with PDA (lambda = 0.5):
auto spec = TrainingSpec::builder()
.pp(pp::pda(0.5))
.build();
// Random forest with uniform variable selection:
auto spec = TrainingSpec::builder()
.size(100)
.pp(pp::pda(0.0))
.vars(vars::uniform(3))
.build();
