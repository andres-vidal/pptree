Detailed Description

Abstract strategy for projection pursuit optimization.

Finds the optimal 1D projection for a given dataset and group partition. Reads from NodeContext: x, var_selection, and the active partition (y or binary_y). Writes: projector (full-dimensional, expanded) and pp_index_value.

pure virtual

Find the optimal projection and store it in the context.

Reads ctx.x, ctx.var_selection, and ctx.active_partition(). Computes the optimal projector on the reduced feature space, expands it back to full dimension via ctx.var_selection.expand(), and writes the result to ctx.projector and ctx.pp_index_value.

Parameters
ctxNode context (reads x, var_selection, active partition; writes projector, pp_index_value).
rngRandom number generator (unused by deterministic strategies).
