Detailed Description

Contiguous-block representation of grouped observations.

Assumes the response vector is sorted so that observations of the same class are contiguous. Stores the start/end row indices of each class block and provides efficient extraction, subsetting, and computation of between- and within-group statistics.

Groups can be hierarchically merged via remap(), which assigns supergroup labels while tracking the original subgroups.
